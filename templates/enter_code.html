🔑 Получение токена VK ID

⚠️ Важно: Не закрывайте эту страницу пока не получите токен! Сессия действительна 1 час.

1Нажмите кнопку авторизации ниже

2Авторизуйтесь через VK

Откроется сайт VK в новой вкладке

3Скопируйте код из адресной строки

После авторизации VK перенаправит на страницу с кодом

https://oauth.vk.com/blank.html?code=vk2.a.xf_...&device_id=...

Нужно скопировать только значение после code= (до &)

4Вставьте код в поле ниже и нажмите "Получить токен"

🔐 Открыть авторизацию VK
💡 Совет: Откройте авторизацию в новой вкладке, скопируйте код и вернитесь на эту страницу для вставки кода.
